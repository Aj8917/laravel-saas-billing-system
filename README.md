# Subscribly App

## 📦 Backend (Laravel)
Located in `/subscribly`

## ⚛️ Frontend (React)
Located in `/subscribly-client`

## 🚀 Getting Started

### Backend
```bash
cd subscribly
composer install
cp .env.example .env
php artisan key:generate
php artisan serve

## Frontend
cd subscribly-client
npm install
npm start
